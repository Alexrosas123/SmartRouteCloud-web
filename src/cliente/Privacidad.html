Política de Privacidad

Esta política de privacidad describe cómo se manejan los datos en la aplicación SmartRoute Cloud, desarrollada para fines académicos en la UPIICSA - IPN para la optimización de procesos logísticos.

2. Recopilación y Uso de la Información

Para el correcto funcionamiento del sistema de segmentación de rutas, la Aplicación puede recopilar:

  • Información de autenticación (correo electrónico).
  • Dirección IP y modelo del dispositivo para fines de seguridad.
  • Datos técnicos necesarios para la ejecución de los algoritmos de optimización.

3. Datos de Ubicación y Logística

SmartRoute Cloud utiliza datos geográficos para:

  • Optimización de rutas: Calcular la distancia eficiente entre los puntos de entrega.
  • Gestión de Unidades: Asignar entregas según la capacidad de carga (Moto/Camión) y ubicación en tiempo real.

4. Procesamiento mediante Inteligencia Artificial

Utilizamos modelos de optimización (K-Means y algoritmos de ruta corta) para procesar datos geográficos de manera anónima, buscando la máxima eficiencia en los tiempos de entrega.

5. Servicios de Terceros

La infraestructura de datos corre bajo los servicios de Google Firebase. Puede consultar su política oficial en: Privacidad en Firebase.
